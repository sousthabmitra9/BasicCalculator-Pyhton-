# Rounded Button Calculator (Tkinter)

A sleek, modern calculator built with **Python Tkinter** featuring **custom rounded buttons** with press animations. Mimics iPhone-style design with smooth interactions, dark theme, and responsive layout.


## Features
- **Custom Rounded Buttons**: Created using `tk.Canvas` with shadow effects and press animation (button sinks on click).
- **Basic Arithmetic**: Supports `+`, `−`, `×`, `÷`, `%`, `+/-`, and decimal input.
- **Real-Time Input**: Entry field updates as you type.
- **Error Handling**: Shows "Error" for invalid expressions.
- **Responsive Layout**: Clean grid-based button arrangement.
- **No External Dependencies**: Pure Tkinter (Python standard library).

## How to Run
1. Ensure Python 3.x is installed.
2. Save the code as `calculator.py`.
3. Run:
   ```bash
   python calculator.py
